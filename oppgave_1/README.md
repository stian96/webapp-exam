

## Getting Started
### 1. Install pnpm:
```bash
    npm install -g pnpm
```

### 2. Install dependencies:
```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
```

### 3. Initialize Database or Seed Data
```bash
    npm run seed
    # or
    yarn seed
    # or
    pnpm run seed
```

### 4. Start the Development Server
```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
